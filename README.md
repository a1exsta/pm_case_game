# PM Case Game

Интерактивная тренинговая веб-игра для Product/Project Manager:

- 5 сценариев с выбором из 4 вариантов
- scoring `0-3` за каждый ответ
- итоговый профиль менеджера
- тёмная адаптивная тема

## Локальный запуск

Откройте файл `index.html` в браузере или запустите простой сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Деплой

Проект статический (один файл `index.html`), подходит для GitHub Pages, Netlify, Vercel.
